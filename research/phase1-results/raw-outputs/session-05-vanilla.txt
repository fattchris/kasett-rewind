### Cost Optimization Decision
| Strategy | Idle Cost | Cold Start |
|---|---|---|
| Always-on (10 tasks) | ~$110/month | None |
| Scale-to-zero | ~$0 idle | ~30 seconds |
| **Warm standby (chosen)** | **~$22/month** | **Eliminated** |

- **Warm standby**: 1 task kept running per region (min_capacity=1), scale-down floor is 1 (not 0)
- Saves **$88/month** vs. always-on
- Scale-to-zero was rejected due to 30-second Fargate cold start (ALB 502 → Lambda → desired_count=1 flow)
- Cost model documented in module README
